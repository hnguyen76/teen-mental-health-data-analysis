# Teen Mental Health Analysis

## Project Overview

This project explores teen mental health patterns using Python, Pandas, Matplotlib, and Seaborn.

The analysis focuses on:
- Social media usage
- Sleep habits
- Stress and anxiety levels
- Depression indicators

The goal is to identify behavioral trends and visualize potential relationships between lifestyle factors and mental health outcomes.

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Analysis Performed

- Data cleaning and preprocessing
- Missing value analysis
- Duplicate detection
- Exploratory data analysis (EDA)
- Correlation analysis
- Data visualization

## Key Insights

- Teens with lower sleep hours tended to show higher depression indicators.
- Stress and anxiety levels demonstrated a positive correlation.
